# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added

- Initial release of the BigQuery Terraform module.
- Dataset creation with configurable location, expiration, and time travel.
- Native table support with time and range partitioning, clustering, and CMEK.
- View and materialized view support.
- External table support for CSV, JSON, Avro, Parquet, and Google Sheets.
- Routine support for UDFs and stored procedures (SQL and JavaScript).
- Row-level security policy support.
- Data Transfer Service configuration.
- Authorized view access configuration.
- Dataset-level IAM access control.
- Comprehensive examples: basic, advanced, and complete.

## [0.1.0] - 2024-01-01

### Added

- Initial development version with core dataset and table functionality.
